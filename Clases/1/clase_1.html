Buscando una funcin cuadrtica

Utilizando el entorno array, logramos crear una lista de nmeros con un orden especfico. De esta forma podemos relacionar dicha lista con una nueva variable y crear una relacin de dependencia con el objetivo de grficar.

In[]:
x=np.array([-3,-2,-1,0,1,2,3])
plt.figure(figsize=(3,3))
plt.plot(x,x**2)
plt.show()

En estos casos, es mejor utilizar la funcin "linspace" dado que de forma automtica se crea un arreglo de nmeros los cuales se pueden utilizar para crear una relacin ms fina.

In[]:
X=np.linspace(-3,3,50)
plt.figure(figsize=(5,5))
plt.plot(X,X**3)
plt.plot(X,X**2)
plt.plot(X,np.sin(X))
plt.show()
In[]:
plt.figure(figsize=(3,3))
plt.plot(X,X**3,color="red",label="$y=x^3$")
plt.plot(X,3*X**2,color="green",label="$y=3x^2$")
plt.legend()
plt.show()
In[]:
plt.figure(figsize=(3,3))
plt.plot(X,X**3,color="blue",label="$y=x^3$")
plt.plot(X,3*X**2,color="pink",label="$y=3x^2$")
plt.xlabel("Eje$X$")
plt.ylabel("Eje$Y$")
plt.title("Miprimergrfico Priscila Merino")
plt.legend()
plt.show()
